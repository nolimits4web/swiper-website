## Frequently Asked Questions

### How do I create a Tinder-like swipe interface?
The free `cards` effect provides the visual card stack. For production Tinder-style interfaces with swipe direction detection (like/dislike), the <a href="https://uiinitiative.com/catalog/tinder-slider" target="_blank">Tinder Slider</a> premium plugin adds the tinderSwipe event with direction callbacks.

### How do I create Instagram Stories?
You can build a basic stories interface with Autoplay, the Cube effect, and custom CSS. For a production-ready solution with progress bars, pause-on-touch, and video duration sync, use the <a href="https://uiinitiative.com/catalog/stories-slider" target="_blank">Stories Slider</a> premium plugin.

### What's the best slider for high-end marketing websites?
The <a href="https://uiinitiative.com/catalog/shaders-slider" target="_blank">Shaders Slider</a> provides WebGL-based transitions that are impossible to achieve with CSS alone. Effects like morph, wave, ripple, and page-curl create memorable experiences.

### How do I create a true 3D carousel?
The free Coverflow effect provides a 3D-like appearance. For true 3D perspective rotation, use <a href="https://uiinitiative.com/catalog/carousel-slider" target="_blank">Carousel Slider</a> or <a href="https://uiinitiative.com/catalog/panorama-slider" target="_blank">Panorama Slider</a>.

### What slider should I use for e-commerce product pages?
Start with the free Thumbs module for thumbnail galleries. For a complete product showcase template, the <a href="https://uiinitiative.com/catalog/fashion-slider" target="_blank">Fashion Slider</a> is designed specifically for e-commerce.

### Can I use premium effects with React/Vue?
Many premium plugins include React and Vue components. For those that don't, converting vanilla JS to React/Vue/Swiper Element is straightforward. <a href="https://uiinitiative.com" target="_blank">UI Initiative support</a> is always happy to help, and <a href="/swiper-mcp">Swiper's MCP</a> can assist with conversions as well.

### What if I don't want to write code?
<a href="https://studio.swiperjs.com" target="_blank">Swiper Studio</a> lets you visually build sliders and export production-ready code for HTML, React, or Vue.
